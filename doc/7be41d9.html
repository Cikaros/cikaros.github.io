mermaid流程图文档

本文最后更新于：2025-03-03T21:37:40+08:00

graph 声明

graph 关键字声明了流程图的走向。

这两种声明了图的方向是从上到下（TD 或 TB）。

graph TD
	start --> stop

这声明了图的方向是从左到右（LR）。

graph LR
    Start --> Stop

流程图方向

可选的流程图方向为：

  • TB-从上到下
  • TD-自顶向下与从上到下相同
  • BT-从下到上
  • RL-从右到左
  • LR-从左到右

节点和形状

节点（默认）

注意 默认会将 id 做为框中的内容

带有文本的节点

你也可以为框中设置自定义的内容，只需使用 id[Text] 语法即可：

graph LR
    id1[This is the text in the box]

节点形状

具有圆边的节点

graph LR
    id1(This is the text in the box)

体育场形状的节点

graph LR
    id1([This is the text in the box])

子例程形状的节点

本网页暂时不支持此节点

圆柱状的节点

graph LR
    id1[(Database)]

圆形式的节点

graph LR
    id1((This is the text in the circle))

非对称形状的节点

graph LR
    id1>This is the text in the box]

目前，只有上面的形状是用的，与它对称的图形暂不可用。未来也许可行！

菱形

graph LR
    id1{This is the text in the box}

六角形节点

graph LR
    id1{{This is the text in the box}}

平行四边形

graph TD
    id1[\This is the text in the box\]

梯形

graph TD
    A[/Christmas\]

梯形 alt

graph TD
    B[\Go shopping/]

节点之间的连接线

节点可以通过连接线连接。可以具有不同类型的连接线，也可以将文本字符串附加到连接线上。

带箭头的直线

graph LR
	A-->B

普通直线

graph LR
	A---B

带文本的直线

graph LR
	A--This is the text---B

或者

graph LR
	A---|This is a text|B

带箭头和文本的直线

graph LR
	A-->|Text|B

或者

graph LR
	A-- Text-->B

虚线

graph LR
	A-.-B

带文本虚线

graph LR
	A-. text.-B

带箭头和文本的虚线

graph LR
	A-.Text.->B

粗直线

graph LR
	A==>B
	B==text===C
	C==Text==>D

多重连线

graph LR
	A--Test-->B--Test-->C

也可以按照以下方式在同一行声明多个节点连接：

graph TB
	A-->B & C -->D
	a & b --> c & d

箭头样式

当声明为 flowchart 而不是 graph 时，支持如下的新箭头：

flowchart LR
	A --o B
	B --x C
	a o--o b
	b x--x c
	c <--> d

具有特殊意义的字符

双引号

可以将文本放在引号中，用于呈现更麻烦的字符。如下例所示：

graph LR
    id1["This is the (text) in the box"]

转义字符

可以使用此处示例的语法对字符进行转义。

graph LR
       A["A double quote:#quot;"] -->B["A dec char:#9829;"]

子图

以下是示例用法：

事件绑定

可以将 click 事件绑定到节点上，单击可能导致 javascript 回调或页面跳转，该跳转将在新的浏览器选项卡中打开。注意：使用 securityLevel='strict' 时将禁用此功能，使用时将启用此功能 securityLevel='loose'。

click nodeId callback

  • nodeId 是节点 id
  • callback 是在显示图形的页面上定义的 JavaScript 函数的名称，该函数会将 nodeId 作为参数回调。

注释

可以在流程图中输入注释，解析器将忽略它们。注释必须自己一行，并且必须以 %%（双百分号）开头。注释开始到下一个换行符之后的所有文本都将被视为注释，包括任何流语法

graph LR
	A -- text --> B{node}
%% 	A -- text --> B -- text2 --> C
